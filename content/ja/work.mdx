# 制作物

これまでに取り組んだプロジェクトの一部をご紹介します。各プロジェクトは、私が長年にわたって習得してきた異なるスキルと技術を表しています。

## 注目のプロジェクト

### [ポートフォリオウェブサイト](#)

Next.js、MDX、Tailwind CSSで構築されたモダンでレスポンシブなポートフォリオウェブサイト。ダークモード、多言語対応、スムーズなトランジションなどの機能を備えています。

**使用技術**: Next.js, TypeScript, Tailwind CSS, MDX

### [Eコマースプラットフォーム](#)

商品カタログ、ショッピングカート、決済統合を備えた本格的なEコマースプラットフォーム。

**使用技術**: React, Node.js, MongoDB, Stripe API

### [AI画像ジェネレーター](#)

テキストプロンプトに基づいてユニークなビジュアルを作成するAI駆動の画像生成ツール。

**使用技術**: Python, TensorFlow, Flask, React
